# Udemig Academy Clone

## 📖 About The Project

This project is a **frontend clone of the Udemig Academy website** created for practice and learning purposes.  
The goal of this project is to improve my **HTML and CSS skills** by recreating a real website interface.

## 🎯 Project Purpose

The main purpose of this project is to practice modern web development techniques and understand how real-world website layouts are structured.

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Responsive Design

## ✨ Features

- Modern and clean UI design
- Responsive layout for different screen sizes
- Hover effects and basic animations
- Semantic HTML structure
